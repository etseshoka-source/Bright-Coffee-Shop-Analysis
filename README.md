# Bright-Coffee-Shop-Analysis
This project analyzes Bright Coffee Shop sales data to discover insights into sales performance, customer behavior and product trends. 

##Aim
To provide the new CEO with an analysis that will assist in making informed decision to improve revenue and operational effeciency.

##Objectives
-To analyze sales distribution per product
-Calculate total revenue and analyze sales perfomance
-Identify the top selling products and the best performing store overall.
-Suggest actionable insights recommendations

####Tools used
-Miro for planning
-Canva for project planning
-SQL(Databricks) for quering and aggregating data
-Excel for data storage and visualisation
